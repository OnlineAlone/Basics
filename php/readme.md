# PHP Modules

```
sudo apt-get install software-properties-common
sudo add-apt-repository ppa:ondrej/php
sudo apt-get update
```


**PHP Modules - v5.6**
```
sudo apt-get install php5.6 libapache2-mod-php5.6 php5.6-common php5.6-soap php5.6-ldap php5.6-imap php5.6-cli php5.6-gd php5.6-json php5.6-mysql php5.6-pgsql php5.6-cgi php5.6-fpm php5.6-gmp php5.6-curl php5.6-mbstring php5.6-intl php5.6-mcrypt php5.6-xml php5.6-zip php5.6-bz2 php-imagick php-apcu php-memcached php-redis php-pear
```
Settings
```
/etc/php/5.6/apache2/php.ini=Configuration for mod_php
/etc/php/5.6/cgi/php.ini=Configuration for scripts run via CGI
/etc/php/5.6/fpm/php.ini=Configuration for scripts run via FPM
/etc/php/5.6/cli/php.ini=Configuration for command-line scripts
```
